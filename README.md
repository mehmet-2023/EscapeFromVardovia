## üöÄ Getting Started

### Prerequisites

- Python 3.8+
- API keys for Groq and Stability AI

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/mehmet-2023/EscapeFromVardovia.git
   cd textadventure
   ```
2. Create and activate a virtual environment:

   ```bash
   python -m venv venv
   source venv/bin/activate  # On Windows: venv\Scripts\activate
   ```
3. Install dependencies:

   ```bash
   pip install -r requirements.txt
   ```
4. Create a `.env` file with your API keys:

   ```
   GROQ_API_KEY=your_groq_api_key_here
   STABILITY_API_KEY=your_stability_ai_key_here
   ```

## üöÄ Getting Started on Windows

### Prerequisites
- Windows 10/11 (64-bit)
- Python 3.8 or higher
- API keys for [Groq](https://console.groq.com/) and [Stability AI](https://stability.ai/)

### Installation Guide for Windows

1. **Download and Install Python** (if not already installed):
   - Visit [Python's official website](https://www.python.org/downloads/windows/)
   - Download the latest Windows installer (64-bit)
   - **IMPORTANT**: During installation, check "Add Python to PATH"
   - Complete the installation

2. **Clone or Download the Repository**:
   - Using Git (recommended):
     ```powershell
     git clone https://github.com/mehmet-2023/EscapeFromVardobia.git
     cd textadventure
     ```
   - Or download as ZIP and extract to your preferred location

3. **Set Up Virtual Environment**:
   Open Command Prompt (cmd) in the project folder and run:
   ```powershell
   python -m venv venv
   .\venv\Scripts\activate
   ```
   You should see `(venv)` at the beginning of your command prompt

4. **Install Dependencies**:
   ```powershell
   pip install -r requirements.txt
   ```

5. **Configure API Keys**:
   - Create a new file named `.env` in the project folder
   - Add your API keys (get them from the respective services):
     ```
     GROQ_API_KEY=your_groq_api_key_here
     STABILITY_API_KEY=your_stability_ai_key_here
     ```
   - Save the file

### Running the Game

Start the web interface:

```bash
python web_interface.py
```

Then open your browser to `http://localhost:5000` to begin your adventure!
